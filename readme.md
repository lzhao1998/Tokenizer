**Tokenizer**
1. Integer
````
   - decimal = 0,1,2,3,4,...
   - binary  = 0,1,10,11,100,...
   - octal   = 1,2,3,4,5,6,7,10,11,...
   - hex     = 0x09,0x0A,0xFF,...
````
2. Floating point
```
  - 0.12e+13
  - 1.013
  - 2.
```
3. String
````
  - "Hello"
  - 'Hello'
  - "E"
  - "a"
````
4. Character
```
  - 'a'
  - 'b'
```
5. Operator
```
  - &&,!,@,||,&@
```
6. Identifier
```
  - _value2_
  - AliBaba
  - _damnDaniel2
```

That's all for my tokenizer. :simple_smile:
